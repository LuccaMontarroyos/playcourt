# PlayCourt – Sistema de Reserva de Quadras Esportivas


Muitas quadras esportivas realizam reservas por meio de aplicativos de mensagens ou ligações telefônicas. Esse processo costuma gerar problemas como conflitos de horários, falta de organização e dificuldade para controle das reservas.
 O PlayCourt surge como uma plataforma digital para facilitar o gerenciamento e o agendamento de quadras esportivas de forma simples e organizada.


### A falta de organização no agendamento de quadras esportivas pode gerar:
- reservas duplicadas
- perda de horários
- dificuldade de controle por parte dos administradores

### O sistema busca centralizar e automatizar esse processo.

## Público-Alvo
- Donos ou administradores de quadras esportivas
- Clientes que desejam reservar horários para prática esportiva
- Funcionários responsáveis pela gestão das reservas
